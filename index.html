开黑召集小工具。群友登录后更新自己的状态（空闲 / 忙碌 / 睡觉 / 自定义）， 同群的人能一眼看到谁在线。当 3 人及以上同时空闲 时，顶部会弹出横幅通知， 一键喊上号。通过邀请码加群，头像 + 昵称区分身份。

下方展示 3 种视觉风格 探索，切换 1 / 2 / 3 可直接换主题。 每屏都是可交互 React 组件，点按可切换状态 / 设置时长 / 触发通知等。
